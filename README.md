## 概要

専門学校在学時より更新を続けている写真のポートフォリオです。学習の素材としても活用していたことから、いくつかのフレームワークで作り直しを行なっています。<br>
どなたかに作品を見ていただくためというより、自分がどんなふうに写真を撮影してきたかを見返すためのサイトです。そのため、写真のジャンル分け等をせず、ビジュアル的な優劣もなく、ログのように全てを確認しやすいデザインにしました。
